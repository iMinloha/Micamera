# 你好~欢迎使用MICAMERA

本产品名就是Micamera，本项目完全开源，包括模型代码单片机等等内容

![logo](img/logo.jpg)

本项目支持4k图片和2k视频，并支持在线管理，实时查看以及容量查询，外壳使用了光固化，总体成本在500元左右。

---

## 项目构成

本项目分为嵌入式端、操作系统端、移动端

- 嵌入式端：stm32文件夹，使用stm32h750xbh6
- 操作系统端：php和python文件夹，分别放在网站目录和用户目录，开机自启
- 移动端：APP文件夹，使用Gradle编译java（而非kotlin）
